# Paketverwaltung

**`apt`**, **`apt-file`**, `dpkg`

/etc/apt/sources.list

---

## [apt](https://wiki.ubuntuusers.de/apt/apt/)

```bash
apt update

apt upgrade

apt search $PAKET

apt install $PAKET
```

## [apt-file](https://wiki.ubuntuusers.de/apt-file/)

```bash
apt-file update

apt-file search $FILE
```

## [dpkg](https://wiki.ubuntuusers.de/dpkg/)

```bash
dpkg -i $DATEI.deb  ## --install
```
